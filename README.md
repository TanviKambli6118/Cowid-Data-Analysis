An interactive data visualization dashboard built using Streamlit, Pandas, and Plotly to analyze global and country-wise COVID-19 data.

📌 Features
📊 Overview Dashboard
Total cases, deaths, and vaccinations
Daily trends (cases & deaths)
Area charts for comparison

🌍 Global Map
Choropleth map of COVID-19 cases worldwide
Animated timeline showing spread over time

💉 Vaccination Analysis
Country-wise vaccination comparison
Daily vaccination trends

🔮 Predictions
7-day moving average for case trends

📈 Correlation Analysis
Heatmap of key features
Bubble chart (cases vs deaths vs population)

🛠️ Tech Stack
Python
Pandas & NumPy – Data processing
Plotly Express – Interactive visualizations
Streamlit – Web app framework

📂 Dataset
Source: Our World in Data (OWID COVID dataset)
File used: owid-covid-data.csv
⚙️ Dashboard Controls

Select country (India, United States, Brazil, United Kingdom)
Choose date range
Navigate through tabs for different analyses

💡 Key Insights
Helps understand COVID-19 trends over time
Compares countries based on cases, deaths, and vaccinations
Visualizes global spread effectively
Provides simple predictions using moving averages
